# BambuMonitor Windows 使用说明

这是一款给 Bambu Lab 打印机用的桌面悬浮监控工具。  
登录后可以自动读取你账号下的打印机，并在桌面顶部显示实时打印进度。

## 运行环境

- Windows 10 或 Windows 11
- 已安装 [Node.js LTS](https://nodejs.org/zh-cn)
- 电脑与打印机在同一局域网内时，实时状态会更完整

## 第一次运行

1. 安装 Node.js
2. 打开项目目录
3. 在地址栏输入 `cmd` 后回车
4. 在终端中执行：

```bash
npm install
npm run electron:dev
```

启动后会弹出登录窗口。

## 登录说明

- 支持账号密码登录
- 如果云端要求安全验证，会自动切到验证码模式
- 登录成功后会读取账号下的打印机列表

## 扫描与连接

- 程序会自动扫描局域网中的拓竹打印机
- 扫描大约需要 6 秒
- 如果某台设备没扫到 IP，可以在悬浮窗里点击“设置IP”手动输入

## 打包成安装版

如果你想生成可分发的 Windows 安装包：

```bash
npm run electron:build
```

完成后可在 `release/` 目录看到打包结果。

## 常用快捷键

- `Ctrl + Shift + L`：锁定/解锁鼠标穿透
- `Ctrl + Shift + H`：切换横向/纵向布局

## 常见问题

### 1. 登录失败

- 检查账号密码是否正确
- 检查网络是否能访问拓竹云端
- 如果提示会话过期，重新登录即可

### 2. 扫描不到打印机

- 确认电脑和打印机在同一个局域网
- 确认打印机已经绑定到当前账号
- 可以尝试手动设置打印机 IP

### 3. 打包后没有图标

当前项目已经包含 Windows 图标资源；如果你替换品牌图标，请同步更新 `build/icon.ico`。
